# HalalCheck – AI Ingredient Classification
**Silver Award – IGIIDeation 2025**

## Overview
HalalCheck is an OCR-based machine learning prototype developed to extract text from food packaging images and classify ingredients as Halal or Haram. The project was awarded Silver at the International Grand Invention, Innovation & Design Expo (IGIIDeation) 2025 and demonstrates the use of AI for practical food verification solutions.

## Project Goal
The goal is to automate ingredient verification, helping users and manufacturers quickly identify Halal compliance from product packaging. This project integrates OCR, NLP, and machine learning to create a robust pipeline for text extraction and classification.

## Features
- **OCR Pipeline**: Uses Tesseract OCR to extract text from food packaging images.
- **Text Preprocessing**: Cleans and prepares text for feature extraction (removing stopwords, special characters, etc.).
- **Machine Learning Classification**:
  - Uses TF-IDF vectorization for feature representation.
  - Implements Logistic Regression to classify ingredients as Halal or Haram.
- **Performance Metrics**: Evaluated using Accuracy, Precision, Recall, and F1 Score, achieving 98% accuracy in experiments.
- **Competition Recognition**: Awarded Silver Medal at IGIIDeation 2025, with the abstract published in the official proceedings.
